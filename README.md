# foldersize
WIP : Just a tool to monitor folder and sub-folder sizes over time. 


## Sample
```
(test) arun@fedora:~$ folmon diff -s 2018-05-05 -e 2018-05-07 -f /log/fruit
+---+-------------------+------------+------------+---------------------+
|   |       folder      | 2018-05-05 | 2018-05-07 | percentage_increase |
+---+-------------------+------------+------------+---------------------+
| 0 |     /log/fruit    |   16.621   |   28.968   |        74.292       |
| 1 |  /log/fruit/apple |   5.573    |   9.773    |        75.371       |
| 2 | /log/fruit/grapes |   5.513    |   9.572    |        73.639       |
| 3 | /log/fruit/orange |   5.535    |   9.624    |        73.856       |
+---+-------------------+------------+------------+---------------------+
(test) arun@fedora:~$ folmon diff -s 2018-05-05 -e 2018-05-10 -f /log/
```
